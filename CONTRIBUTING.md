# Contributing to PromptPerfect

> This document is being filled out. Check back soon.

## Getting Started
- Clone the repo
- Copy .env.example to .env.local and fill in values
- Run npm install
- Run npm run dev

## Branch Naming
Use format: PP-{ticket-number}/{short-description}
Example: PP-501/repo-cleanup

## Pull Requests
- All PRs must target main
- CI must pass before merge
- At least 1 reviewer required
